# homebrew-brews-ha

Homebrew tap for [Hosting Accessories](https://github.com/hostingaccessories) projects.

## Installation

```
brew tap hostingaccessories/brews
```

## Available Formulae

- **[bubble-bot](https://github.com/hostingaccessories/bubble-bot)** - Ephemeral containers for AI agents

## Usage

```
brew install hostingaccessories/brews/bubble-bot
```
